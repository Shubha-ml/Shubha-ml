## Selected Projects

### Formal Modeling and Verification Framework

A unified framework for specifying and verifying safety-critical systems using TLA+.
Includes case studies such as traffic systems, railway signaling, and autonomous vehicles.

### UML → TLA+ Web Application

A web-based tool that converts UML diagrams into TLA+ specifications to enable early-stage verification of system behavior.

### TLA+ → Python Code Generator

A translation pipeline that converts formal specifications into executable Python code with embedded runtime checks.

### Automata to Formal Models

Transformation of DFA/NFA models into TLA+ specifications for formal analysis and validation.
